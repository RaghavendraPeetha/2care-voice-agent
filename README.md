# 2Care Voice AI Agent

## Overview

AI-powered hospital receptionist for Yashoda Hospital.

Supports:
- Appointment booking
- Rescheduling
- Cancellation
- Appointment lookup
- Emergency detection

---

## Architecture

Voice Platform
↓
DeepAgents

LangChain Tools
↓
FastAPI Backend
↓
SQLite Database
